Audio Effect Similarity Demo

Wright, A., Välimäki, V., Juvela, L. — Journal of the Acoustical Society of America, 2024

About the Metric

This demo accompanies a study on objective metrics for evaluating neural audio effect models. Each example below shows a short clip processed through a neural audio effect emulator, alongside the original input and the reference recording (the real hardware device). Listen to all three and judge how similar the model output is to the reference — then compare your impression to the metric score.

The metric used here is the MFCC-L1 distance: the L1 distance between the Mel-Frequency Cepstral Coefficient (MFCC) sequences of the model output and the reference. Smaller values indicate greater similarity.

A piecewise linear model (fitted to perceptual listening test data) converts the MFCC-L1 distance into a predicted perceptual rating on a 0–100 scale:

x = log10(MFCC-L1)
rating = 100                   if x ≤ 0.464
rating = 100 − 69.4 × (x − 0.464)  otherwise

A rating of 100 means the metric predicts the output is perceptually indistinguishable from the reference; lower values indicate increasing audible degradation.
